    • About

    Foundation is one of many front-end frameworks you can use to build your website. You can access and modify a library of standardized CSS components, rather than building everything from scratch. This site showcases some of the most basic features of Foundation, with instructions to help you get started using it for your own projects. Some additional resources for more detailed documentation are included as well, once you've exhausted what we have on our site!

    Before we get started, be sure to keep the source code handy!

    Each feature shown will include a reference to the specific lines in which it's implemented.
    Right click and select View page source to open the source code in a new tab.

    Setup

    The fastest way to start using the Foundation features showcased on this site is to simply add the CDN link into your HTML head section.

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.0/dist/css/foundation.min.css">

    Alternatively, download the complete version here for all the features available in the framework.

    XY Grid

    An XY grid system is used in Foundation to separate a webpage into grids, and grids into cells and cell-blocks. In this site, the menu and content are separated into two y-cell blocks, and both blocks are contained in a grid-y. The heading is in a separate grid-x.
